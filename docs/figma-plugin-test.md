# Figma Plugin Test Guide

API keyなしで、FigmaローカルプラグインとしてA/B/C/Dの流れを最後まで確認する手順です。

## 1. ビルド

```bash
npm install
npm run build
```

## 2. Figmaローカルプラグインとして読み込む

Figma Desktopを開きます。

```text
Plugins > Development > Import plugin from manifest...
```

読み込むファイル:

```text
C:\Users\orimo\OneDrive\ドキュメント\FigmaPJ\manifest.json
```

起動:

```text
Plugins > Development > AI Creative Assistant Foundation
```

## 3. APIなしでDemoフローを確認する

1. 探索画面を開く
2. 「Demoフローを開始」を押す
3. 「探索を開始」を押す
4. 5つの方向性とSVG previewが表示されることを確認する
5. 「5案をまとめてFigmaに配置」を押す
6. Figmaキャンバス上に5案が横並びで配置されることを確認する

生成されるフレーム名の例:

- `AI_SEMINAR_problem_01`
- `AI_SEMINAR_benefit_02`
- `AI_SEMINAR_practical_03`
- `AI_SEMINAR_trust_04`
- `AI_SEMINAR_beginner_05`

## 4. 診断をテストする

診断は1案だけを読む工程です。

1. Figmaキャンバス上で、診断したいバナー案を1つクリックする
2. プラグインの診断画面へ移動する
3. 「選択中のフレームを診断」を押す
4. 診断概要、最初に伝わること、強い点、気になる点、最初に直すなら、派生案が表示されることを確認する
5. 必要に応じて「診断ボードをFigmaに追加」を押す

1つだけ選択する理由:

診断は、1案の伝わり方や改善点を深く読む工程です。複数案を比べる場合は比較画面を使います。

## 5. 比較をテストする

比較は複数案の役割や強みを整理する工程です。

1. Figmaキャンバス上で、比較したいバナー案を2〜5個選択する
2. プラグインの比較画面へ移動する
3. 「選択中の案を比較」を押す
4. 比較概要、比較表、ベース候補、次点候補、background briefが表示されることを確認する
5. 「この案で背景を仕上げる」を押して仕上げ画面へ進む

複数選択する理由:

比較は、案ごとの役割、強み、懸念、向いている用途を並べて判断する工程です。1案だけでは比較できないため、2案以上を選択します。

## 6. 仕上げをテストする

仕上げは、比較で選ばれたPrimary案に背景を適用する工程です。

1. 仕上げ画面で対象フレーム名とbackground briefを確認する
2. 背景スタイル候補を選ぶ
3. 「Demo背景を生成」を押す
4. 生成された背景の色やスタイルを確認する
5. 「背景をFigmaに適用」を押す
6. Figmaキャンバス上の対象フレームに背景レイヤーが追加されることを確認する

確認ポイント:

- テキストやCTAが消えていない
- 背景レイヤーだけが追加されている
- 対象フレーム名が比較で選ばれたPrimary案と一致している

## 7. よくあるエラーと対処

### フレームが選択されていません

Figmaキャンバス上でバナー案のFrameをクリックしてから、もう一度実行してください。

### 複数選択されています

診断では1つだけ選択してください。複数案を見たい場合は比較画面を使います。

### Frame以外を選択しています

テキストや図形ではなく、バナー案全体のFrameを選択してください。

### 比較には2つ以上のフレームが必要です

Figmaキャンバス上で2〜5個のバナー案を選択してください。

### APIが未設定です

Demo Modeへfallbackします。API keyなしでも探索、診断、比較、背景適用まで確認できます。

### demo fallback中です

UIに「Demo Modeに切替」や「API未設定でもDemo診断」と表示されます。live providerを使う場合は `src/config/apiSettings.ts` と `src/config/providers.ts` を確認してください。

### background対象フレームが見つかりません

比較後にFigma上の対象フレームを削除した可能性があります。もう一度探索で配置し、比較から仕上げへ進んでください。

## 8. Webローカルで確認できる範囲

- UI表示
- demoデータ
- SVG preview

## 9. Figma内でしか確認できない範囲

- SVGをFigmaに配置
- 選択フレーム診断
- 複数フレーム比較
- 背景適用
- Figmaプロセスボード生成
