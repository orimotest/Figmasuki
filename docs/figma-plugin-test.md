# Figma Plugin Test

## 基本確認

1. `npm run build`
2. Figmaで `manifest.json` を読み込む
3. プラグインを起動
4. 探索タブで「noteサンプルを読み込む」
5. 「探索を開始」
6. SVG候補が5案出ることを確認
7. 「5案をまとめてFigmaに配置」
8. 「プロセスボードをFigmaに作成」

## 診断

1. Figma上で800x450の候補フレームを1つ選択
2. 診断タブで「選択中のフレームを診断」
3. 診断結果がカードで表示されること
4. 「診断ボードをFigmaに追加」

## 比較

1. Figma上で候補フレームを2から5個選択
2. 比較タブで「選択中の案を比較」
3. 比較表、ベース候補、次点候補、background briefが表示されること
4. 「この案で背景を仕上げる」

## 仕上げ

1. 仕上げタブへ移動
2. 背景スタイル候補を選択
3. 「背景を生成」
4. 「背景を適用」
5. 「仕上げボードをFigmaに追加」
