# Figma Process Board

`一連のプロセスをFigmaに配置` を押すと、Figmaキャンバス上に横長のプロセスボードが生成されます。

## 生成されるボード

- Project Header
- 30 Ideas Explore
- 15 Typography Drafts
- 5 Refined SVGs
- Diagnosis
- Compare
- Background Variations
- Final Candidate

## 重要な方針

Figmaには完成物だけでなく、AIが考えた過程を残します。

- 30案探索ではSVGを置かず、コピーと訴求軸を記録します。
- 15案ドラフトでは文字組みSVGを小さく整列します。
- 5案高品質SVGでは比較できる代表案を記録します。
- 比較、背景、最終案は後続工程の判断材料として残します。
