# Demo Flow

## note demo

1. 探索タブ
2. 「noteサンプルを読み込む」
3. 「探索を開始」
4. コピー方向性とSVG候補を確認
5. 「生成してFigmaに配置」

## seminar demo

1. 探索タブ
2. 「セミナーサンプルを読み込む」
3. 「探索を開始」
4. 「生成してFigmaに配置」
5. 2から5案を選択して比較
6. ベース候補から仕上げへ進む
7. 背景を生成して適用

## デモで見せるポイント

- UI内で現在工程が分かる
- Figma上にプロセスボードが残る
- SVG候補だけでなく、判断材料もレビューできる
- 完成案だけではなく、思考プロセスが説明できる
