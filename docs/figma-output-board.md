# Figma Output Board

Figmaには最終案だけではなく、AIが整理したプロセスをボードとして出力します。

## 構成

1. Project Header
2. Copy Exploration Board
3. Layout Strategy Board
4. Layout Candidates Board
5. Diagnosis Board
6. Compare Board
7. Finish Board

横方向に流れるワークスペースとして配置されます。

## Project Header

プロジェクト名、用途、サイズ、入力タイプ、要件要約、ターゲット、ゴール、生成日時、provider modeを表示します。

## Copy Exploration Board

30案を探索し、5方向へ抽出したコピー方向性をカードで表示します。

## Layout Strategy Board

各方向のlayout type、構図、優先順位、色、フォント、背景方針を表示します。

## Layout Candidates Board

5つのSVG候補をカード内に整列して配置します。方向名、短い説明、direction idも表示します。

## Diagnosis Board

診断概要、最初に伝わること、強い点、気になる点、最初に直すなら、派生案を表示します。

## Compare Board

比較概要、比較表、ベース候補、次点候補、選定理由、background briefを表示します。

## Finish Board

対象案、background brief、背景スタイル、適用前、適用後、最終案の状態を表示します。
