# Demo Data Spec

## note_thumbnail Demo

The note thumbnail demo targets an article about design thinking and creative workflow in the AI era. It should eventually include five distinct directions covering question-led, editorial, practical, critical, and quiet-statement approaches.

## seminar_banner Demo

The seminar banner demo targets an introductory AI seminar or webinar. It should eventually include five distinct directions covering problem empathy, benefit-first messaging, practical know-how, trust, and beginner-friendly framing.

## Direction Count

Each content type should provide five demo directions. The foundation phase may include a smaller seed set, but the data shape is prepared so five directions can be added without changing workflow or UI contracts.

## SVG Replacement

Demo SVGs are intentionally lightweight and stored separately from workflow logic. They should be replaced later with higher-quality SVG samples while preserving the same `SvgCandidate` structure and 800x450 canvas contract.
